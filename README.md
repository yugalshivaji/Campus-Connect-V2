# CampusSphere - Event Management & Participation Platform

CampusSphere is an immersive, 3D-powered platform designed for college campuses to manage events, societies, and student participation.

## 📱 Platform Overview
- **Mobile-First**: Designed for a premium mobile experience.
- **Immersive UI**: 3D backgrounds, glassmorphism, and smooth animations.
- **Role-Based**: Student, Society Organizer, and Admin roles.

## 🚀 Core Features
1. **Event Discovery**: Personalized feed with category filters and trending events.
2. **3D Interactive Cards**: Engaging event cards with tilt and motion effects.
3. **Smart Calendar**: Timeline view of registered events with AI-powered slot analysis.
4. **QR Systems**: 
   - **Attendance**: Unique QR per registration for secure check-ins.
   - **Food Coupons**: One-time use QR coupons for event refreshments.
5. **Society Ecosystem**: Profiles, memberships, and follow systems.
6. **AI Recommendations**: Gemini-powered event suggestions based on user interests.
7. **Gamification**: Points, badges, and achievement tracking.
8. **Dynamic Themes**: Light, Dark, and Neon modes.

## 🛠️ Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Motion, React Three Fiber.
- **Backend**: Node.js, Express.
- **Database**: Firebase Firestore.
- **Auth**: Firebase Auth (Google Login).
- **AI**: Google Gemini API.

## 📦 Deliverables
- **Architecture**: Full-stack Express + Vite + Firebase.
- **Schema**: Detailed Firestore collections in `firebase-blueprint.json`.
- **APIs**: REST endpoints in `server.ts`.
- **Logic**: QR validation and payment verification placeholders in backend.
- **Security**: Robust Firestore rules in `firestore.rules`.
